Plasma Mobile Nightly on Alpine

Development builds of Plasma Mobile packages for Alpine Linux, built every night from git. Easily develop and test KDE packages on postmarketOS.

Available as an Alpine Linux repository, and as pre-built postmarketOS images for the various devices (including the PinePhone).

Warning

These packages and images will be broken! These are not official builds, and Sineware provides this service for development and testing convenience.

Please, do not pester KDE/postmarketOS/etc devs when things break! Issues can be reported to our discord first.

Converting an existing postmarketOS Install

Add the repository URL to the /etc/apk/repositories file and the key to the /etc/apk/keys folder.

Then run apk update && apk upgrade and your installed packages will upgrade to the development builds.
